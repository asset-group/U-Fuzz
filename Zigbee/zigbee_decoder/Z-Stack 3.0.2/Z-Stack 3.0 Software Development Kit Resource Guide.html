SimpleLink™ ZigBee® Wireless MCU

Z-Stack™ 3.0 Software Development Kit Resource Guide

Version: 3.0.2
May 10, 2017

The Texas Instruments Z-Stack™ 3.0 software development kit (SDK) includes all the necessary software to get started on the development of ZigBee® 3.0 applications using the SimpleLink portfolio of Wireless MCUs from Texas Instruments (CC2530/1, and CC2538).

It includes object code with the ZigBee ® protocol stack, sample projects and applications, and several tools for the design and development of applications powered by ZigBee ® technology.

In addition to the software, the kit contains the following documentation and tools to aid in development:
